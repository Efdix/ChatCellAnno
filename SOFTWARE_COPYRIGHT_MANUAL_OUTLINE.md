## 第一章 软件概述
### 1.1 编写目的
[本说明书旨在指导用户安装、配置及使用 ChatCellAnno 软件，完成单细胞测序数据的自动化细胞类型注释。]
### 1.2 软件背景
[针对生物信息学单细胞分析中，自动化注释工具复杂、依赖 API 且隐私受限的问题，ChatCellAnno 提供了一种基于剪贴板交互的“零 API”隐私安全注释方案。]
### 1.3 核心术语与定义
- **Marker Genes**: 显著差异表达基因。
- **Zero-API (零 API)**: 不直接通过网络代码调用模型，而是通过系统剪贴板进行数据流转。
- **Prompt Engineering (提示词工程)**: 优化发送给 AI 的指令以获得更精准的生物学结论。

### 2.1 硬件要求
- 处理器：Intel Core i5 或以上。
- 内存：8GB RAM 或以上。

- 操作系统：Windows 10/11 64-bit。
